## :notebook: Allnotar API

Allnotar - The place where you can write all your notes.

## :ballot_box_with_check: Installation

```bash
$ yarn install
```

## :play_or_pause_button: Running the app

```bash
# Development
$ yarn start

# Watch mode
$ yarn start:dev

# Production mode
$ yarn start:prod
```

## :test_tube: Test

```bash
# Unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# Test coverage
$ yarn test:cov
```
